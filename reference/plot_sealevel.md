# Download and plot essential climate data

Plots the global mean sea level data retrieved using
[`get_sealevel()`](get_sealevel.md) with ggplot2. The output ggplot2
object may be further modified.

## Usage

``` r
plot_sealevel(dataset = get_sealevel(), print = TRUE)
```

## Arguments

- dataset:

  Name of the tibble generated by `get_sealevel`, defaults to calling
  `get_sealevel`

- print:

  (boolean) Display sealevel ggplot2 chart, defaults to TRUE. Use FALSE
  to not display chart.

## Value

Invisibly returns a ggplot2 object with sealevel chart

## Details

`plot_sealevel` invisibly returns a ggplot2 object with a pre-defined
sealevel change chart using data from `get_sealevel`. By default the
chart is also displayed. Users may further modify the output ggplot2
chart.

## Examples

``` r
# \donttest{
# Fetch sealevel data:
gmsl <- get_sealevel()
#
# Plot output using package's built-in ggplot2 defaults
plot_sealevel(gmsl)


# Or just call plot_sealevel(), which defaults to get_sealevel() dataset
plot_sealevel()


p <- plot_sealevel(gmsl, print = FALSE)
# Modify plot such as: p + ggplot2::labs(title='Rising Waters') # }
```
