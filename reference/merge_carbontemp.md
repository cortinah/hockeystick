# Download and plot essential climate data

Merge NOAA carbon and NASA temperature datasets on common dates.

## Usage

``` r
merge_carbontemp(carbon = get_carbon(), temp = get_temp())
```

## Arguments

- carbon:

  Name of the tibble generated by `get_carbon`

- temp:

  Name of the tibble generated by `get_temp`

## Value

Invisibly returns a tibble with merged datasets from `get_carbon` and
`get_temp` functions.

## Details

`merge_carbontemp` invisibly returns a tibble with the merged data from
from `get_carbon` and `get_temp` functions. Tibble only includes data
from dates when both datasets are available, essentially from 1960.
